3. Windows Setup Instructions

If Hindi/Marathi voices don't appear:

  1. Open Windows Settings (Win + I)
  2. Click "Time & Language" → "Language & region"
  3. Click "Add a language"
  4. Search for "Hindi" or "Marathi"
  5. Select it and click "Next"
  6. Check "Install language pack" and "Text-to-speech"
  7. Click "Install"
  8. Restart your browser after installation

Note: Windows does NOT include Marathi text-to-speech voice by default. Our app will use Hindi voice as a fallback for Marathi text.

4. Alternative Solutions for Marathi

Since Windows doesn't have built-in Marathi TTS, you can:

  • Option 1: Use the Hindi voice fallback (automatic in our app)
  • Option 2: Install Google Chrome and use Google Translate TTS extension
  • Option 3: Install third-party Marathi TTS software like "eSpeak" or "Festival"
  • Option 4: Use Google Cloud Text-to-Speech API (requires setup)
